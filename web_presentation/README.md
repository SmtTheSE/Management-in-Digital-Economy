# Digital Leadership and Motivation for High-Performing Teams - Web Presentation

This web-based presentation replicates the functionality of PowerPoint with Apple's clean, minimalist design aesthetic. It contains 16 slides covering all assignment requirements for "Digital Leadership and Motivation for High-Performing Teams", enhanced with visualizations and online images.

## Features

- 16 slides matching the assignment content
- Apple-inspired minimalist design with clean typography and ample whitespace
- Responsive layout that works on desktop and mobile devices
- Keyboard navigation (arrow keys, Enter, Space)
- Touch swipe support for mobile devices
- Slide counter and navigation controls
- Speaker notes accessible below each slide
- Visualizations with online images for each topic
- Thematic icons for each concept using Font Awesome
- Smooth animations and transitions

## How to Use

1. Open the `index.html` file in any modern web browser
2. Use the "Next ›" and "‹ Previous" buttons to navigate
3. Use keyboard arrows to move between slides:
   - Right arrow or Enter/Space: Next slide
   - Left arrow: Previous slide
4. On mobile devices, swipe left/right to change slides
5. Speaker notes are displayed below each slide content
6. Visualizations are presented as online images relevant to each topic

## Assignment Requirements Covered

This presentation addresses all assignment requirements:

- Comparison of digital-led management approaches (systems, contingency, behavioural, decision-theory)
- Analysis of digital leadership styles (transformational, participative, visionary)
- Application of motivational theories (Maslow, Herzberg, Vroom)
- Integration of reflective models (Gibbs, Schön)
- Evidence-based recommendations for digital team leadership

## Netlify Deployment

This presentation is optimized for deployment on Netlify:

1. **Direct Deployment**: Simply drag and drop the `web_presentation` folder to Netlify
2. **Git Integration**: Connect a Git repository containing these files to Netlify
3. **No Build Step Required**: This is a static site with no build process needed

### Deployment Features:
- All images are loaded from external sources (no asset management needed)
- SVG visualizations are embedded directly in HTML
- All functionality works without server-side processing
- Responsive design works on all devices
- Optimized for fast loading on the global CDN

### Custom Domain Setup:
1. In Netlify dashboard, go to "Domain settings"
2. Add your custom domain
3. Update DNS records as instructed by Netlify
