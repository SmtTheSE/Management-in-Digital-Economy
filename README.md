# Management in Digital Economy Assignment
## Digital Leadership and Motivation for High-Performing Teams

### Assignment Overview
This assignment addresses challenges of low motivation and inconsistent leadership in a scaling tech start-up through digital-led management and leadership practices.

### Required Materials
- PowerPoint presentation (10-15 slides)
- Speaker notes for each slide
- Individual reflective statements (300 words each)
- References in Harvard style

### Contents
1. [Presentation Outline](presentation/presentation_outline.txt) - Detailed structure with speaker notes
2. [Reflective Statements](reflective_statements/reflective_statements.txt) - Individual reflections for each group member
3. [README.md](README.md) - This file with instructions

### Instructions for Creating the PowerPoint Presentation

#### Step 1: Create the Presentation
1. Open Microsoft PowerPoint
2. Create a new presentation with 10-15 slides
3. Use the [presentation_outline.txt](presentation/presentation_outline.txt) file as a guide for slide content
4. Each slide in the outline corresponds to one PowerPoint slide
5. Include visuals, charts, or infographics to support key arguments
6. Add speaker notes to each slide using the notes provided in the outline

#### Step 2: Include Visual Elements
- Add relevant diagrams comparing management approaches
- Include infographics showing leadership styles
- Add charts illustrating motivational theories
- Insert visual models of reflection processes
- Include screenshots or examples from real-world digital organizations

#### Step 3: Add References
- Include Harvard-style references on the final slide
- Add in-text citations where theories or examples are discussed
- Ensure all sources mentioned in speaker notes are properly cited

#### Step 4: Prepare Individual Reflective Statements
1. Each group member should personalize one of the templates in [reflective_statements.txt](reflective_statements/reflective_statements.txt)
2. Replace placeholder names with actual group member names
3. Adjust content as needed based on individual contributions
4. Ensure each statement is approximately 300 words
5. Maintain the Gibbs' Reflective Cycle structure (Description, Feelings, Evaluation, Analysis, Conclusion, Action Plan)
